🤖 Arxiv Telegram Bot for hep-th

A Python-based Telegram bot that automatically scrapes and posts new arXiv papers from any category (default: hep-th). The bot runs daily using GitHub Actions and sends notifications to a Telegram channel.

Features:

  • Automated daily scraping of new arXiv papers
  • Smart caching to avoid duplicate posts
  • Works with any arXiv category (easily configurable)
  • Integrates with INSPIRE-HEP author links
  • GitHub Actions workflow for scheduled daily runs
  • Minimal dependencies (requests + beautifulsoup4)
